# Ardapoeia

A walkable map of Middle‑Earth set during the Third Age, playable in your browser.

> **Note:** This project is in early development. Expect breaking changes. Contributions are encouraged! :)

_Based on the works of J. R. R. Tolkien. This is an unofficial fan project and is not in any way affiliated with or endorsed by the Tolkien Estate._

**Version:** 0.1.0\
**License:** MIT

- [docs/CITE.md](docs/CITE.md): list of sources

Only [src/](src/) and [web/](web/) are compiled into the final build. [tools/](tools/) is used to generate assets and is not included in the output.
